# CLAUDE.md

This document explains how AI tools were used during the development of the **Domain Records Explorer** project.

## AI Tools Used

The following AI tools were used as development assistants:

* ChatGPT (OpenAI)
* IDE AI autocomplete (e.g., GitHub Copilot) for small boilerplate suggestions

These tools were used to improve productivity while all engineering decisions and final code ownership remained manual.

## Tasks Assisted by AI

AI tools assisted with:

* Brainstorming project architecture and component structure
* Generating initial Vue 3 component and composable scaffolding
* Exploring UI patterns for the domain details view
* Drafting project documentation (`README.md`, `AI_USAGE.md`)
* Discussing testing approaches and potential edge cases

AI output was treated as a **starting point** and not used directly without review.

## Modifications and Corrections

AI-generated suggestions were reviewed and adjusted to:

* Follow Vue 3 Composition API and `<script setup>` best practices
* Fit the chosen project structure and separation of concerns
* Simplify overly complex suggestions
* Maintain minimal dependencies as required by the exercise
* Improve naming, readability, and maintainability

## Validation

AI-generated output was validated through:

* Manual review of all generated code
* Writing tests for core functionality

## Ownership

AI tools were used as assistants, but all final implementation decisions, code integration, and validation were performed by the developer.
